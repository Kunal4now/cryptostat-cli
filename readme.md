# Cryptostat-CLI

Cryptostat is a Node.js based CLI that gets you the real-time stats of your favorite cryptocurrency.

## Installation

Use the npm package manager to install Cryptostat.

```bash
npm install -g cryptostat
```

## Usage
In your terminal enter

```bash
cryptostat
```
